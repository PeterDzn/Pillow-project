# Pillow-project
Pillow 🌙
Aplicativo Desktop Inteligente de Monitoramento e Análise do Sono

O Pillow é um aplicativo desktop moderno focado em monitoramento, visualização e análise inteligente da qualidade do sono.
O projeto combina uma experiência visual premium com recursos voltados para saúde e bem-estar, permitindo que o usuário compreenda melhor seus hábitos de sono através de dashboards interativos e relatórios detalhados.

✨ Funcionalidades
💤 Monitoramento do Sono
Registro manual das sessões de sono
Controle de horário de dormir e acordar
Registro de interrupções durante a noite
Registro de fatores externos (barulho, desconforto, etc.)
📊 Análises Inteligentes
Estimativa da qualidade do sono
Estimativa de sono REM (Rapid Eye Movement)
Análise de sono profundo
Análise de regularidade do sono
Cálculo de eficiência do sono
Insights e recomendações personalizadas
📂 Importação de Dados de Saúde

Compatível com importação de:

Apple Health (export.xml)
Arquivos JSON
Arquivos CSV

O sistema converte os dados importados para uma estrutura interna padronizada para análise.
